Oi! Meu nome é João Vitor Krahembuhl dos Santos. Sou estudante e tenho me dedicado a aprender análise de dados, especialmente com R. Gosto de entender como os dados contam histórias e ajudam a tomar decisões mais informadas. Neste portfólio, compartilho um projeto prático que fiz como parte de uma disciplina de estatística, onde analisei dados de acidentes no Brasil usando bibliotecas como dplyr e ggplot2.

Projeto: Análise de Acidentes com R

  • Importação e exploração de dados com read.csv(), summary() e str()
  • Manipulação com a biblioteca dplyr (filtro, agrupamento, sumarização)
  • Cálculos probabilísticos com base nas variáveis
  • Visualização com a biblioteca ggplot2
